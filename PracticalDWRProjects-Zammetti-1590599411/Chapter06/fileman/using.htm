Using DWR File Manager


Using the DWR File Manager (Fileman) is very much like using the ubiquitous Windows Explorer. You can see a list of roots (or drives) available on the server's file system on the left. You can expand the branches of this directory tree, or click on a directory to view its contents.

On the right is the contents of the selected directory. You will see both files and directories here. For files, you will see its type, if Fileman recognized it, its size, and the date/time it was last modified. For directories, the size is always 0 and the type is always "Directory".

On the right you can select a single file or directory to operate on. When you do, the following toolbar icons are available:

Edit the selected file.
Delete the selected file or directory.
Copy the selected file or directory to the clipboard for later pasting. The original file or directory will not be deleted when pasted.
Cut the selected file or directory to the clipboard for later pasting. The original file or directory WILL BE DELETED when pasted.
Paste the file or directory currently on the clipboard, if any, to the currently selected directory.

The following toolbar icons are available whether a file or directory is currently selected on the right or not:

Move up one level, i.e., the parent of the currently selected directory, if there is a parent.
Refresh the current directory. Note this does nothing if no file system root is yet selected.
Create a new file in the currently selected directory.
Create a new directory in the currently selected directory.
Upload a file. File will be saved to the currently selected directory.
Download the currently selected file.

Note that all of these functions are also available from the menu bar. Speaking of the menu bar, the following addition functions are available there (these are not available on the toolbar):
  • Tools... Print Directory Contents - Provides a printable version of the file grid on the right.
  • Help... Using Fileman - Views this help file.
  • Help... About - Shows the About Fileman popup dialog.
To rename a file or directory, simply double-click the item to rename in the file grid, or click F2 while the row is selected, and you will be able to edit it in-place. The change occurs when you press Enter.

Lastly, the file grid can be sorted by clicking any of the column headers.

That's about it. Have a pleasent day!
